Geely Future Drive System

Interface Design

“Drive together” is an interactive driving system based on the concept of future driving, which include three main parts: 1. “my car”, can connect users and car via app. 2.“drive together”, which can build a connection between cars. 3. “help”, which enables users to seek help in emergency.

“同行”是基于“未来出行”设计的人，车互联交互系统。主要从三个方面出发进行概念设计：1.“我的车”，通过车联网与手机软件平台，实现人与车的连接。2.“车队出行”，在同行的过程中车与车的连接，实现车辆行驶状况的实时共享。3.“帮助”，实现人与人的连接，出现紧急情况时可以寻求帮助。
